## 下一步

本节列出了您现在可以从这里采取的一些潜在的后续步骤。随意探索这些内容或阅读[简介](../../../../concepts/overview/what-is-prisma)页面以获取对 Prisma 的高级概述。

### 继续探索 Prisma Client API

您可以使用Prisma Client API发送各种查询。查看[API参考](../../../ concepts/components/prisma-client)，并使用本指南中的现有数据库设置来尝试它们。

<Tip>

您可以使用编辑器的自动完成特性来了解不同的API调用及其参数。自动完成通常通过按键盘上的<kbd>CTRL</kbd>+<kbd>SPACE</kbd>来调用。

</Tip>

<details><summary>展开以获取更多 Prisma Client API 示例</summary>

以下是您可以通过Prisma客户端发送的更多查询的一些建议：

**筛选所有包含`"hello"`的`Post`记录`**

```js
const filteredPosts = await prisma.post.findMany({
  where: {
    OR: [{ title: { contains: 'hello' } }, { content: { contains: 'hello' } }],
  },
})
```

**创建一个新的`Post`记录，并将其连接到一个现有的`User`记录**

```js
const post = await prisma.post.create({
  data: {
    title: 'Join us for Prisma Day 2020',
    author: {
      connect: { email: 'alice@prisma.io' },
    },
  },
})
```

**Use the fluent relations API to retrieve the `Post` records of a `User` by traversing the relations**

```js
const posts = await prisma.profile
  .findUnique({
    where: { id: 1 },
  })
  .user()
  .posts()
```

**Delete a `User` record**

```js
const deletedUser = await prisma.user.delete({
  where: { email: 'sarah@prisma.io' },
})
```

</details>

### Explore the data in Prisma Studio

Prisma Studio是数据库中数据的可视化编辑器。

你可以用两种方式运行它:

1. 在终端上运行 `$ npx prisma studio`。
2. 从安装程序安装[桌面应用程序](https://github.com/prisma/studio/releases)。支持Windows、macOS、Linux操作系统。

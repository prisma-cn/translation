<TopBlock>

Prisma 是一个数据库工具包，它简化了数据库访问。你可以使用 Prisma 工具来构建 GraphQL 或 REST API，或者作为全栈应用程序的一部分 - 你可以自行决定使用 Prisma 到哪个程度。

<Tip>

If you are building a GraphQL API, check out [GraphQL Nexus](https://nexusjs.org/) - code first GraphQL schemas with [a plugin specifically for Prisma](https://nxs.li/plugins/prisma).

如果你正在构建 GraphQL API，请查看 [GraphQL Nexus](https://nexusjs.org/) - 用 [专用于 Prisma 的插件](https://nxs.li/plugins/prisma) 编写的 code first GraphQL 服务。

</Tip>

</TopBlock>

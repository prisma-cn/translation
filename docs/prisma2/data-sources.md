你可以用任何库和服务的读取或写入来当作 prisma 的数据源。

比如说:

- 数据库
- GraphQL APIs
- REST APIs
- 第三方 APIs (例如：微信 API)

你也可以为数据源实现自己的连接器。后面发布相关信息。

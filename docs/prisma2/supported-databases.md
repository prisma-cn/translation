# Supported Databases

Prisma 2 currently supports the following databases:

| Database | Version | 
| --- | --- |
| PostgreSQL |  9 |
| PostgreSQL |  10 |
| PostgreSQL |  11 |
| PostgreSQL |  12 |
| MySQL | 5.7 | 
| MySQL | 8 | 
| MariaDB | 10 |
| SQLite | 3.28.0 |


Note that a fixed version of SQLite is shipped with every Prisma 2 release.
